# Project skills (agent-agnostic)

Mirrors `.claude/skills/` for any harness that reads the `.agents/skills/<name>/SKILL.md`
convention instead of Claude Code's `.claude/skills/` path. See `.claude/skills/README.md`
for full source attribution, licenses, and the vendored skill list — this folder is kept
in sync with that one manually (no symlinks/submodules).
